# Project Overview

## Problem Statement
Retail companies struggle to track:
- Sales performance
- Inventory levels
- Regional trends

## Solution
We build a dashboard that shows:
- Sales insights
- Inventory status
- Regional performance

## Stakeholders

### Store Manager
Needs:
- Daily sales
- Top products

### Inventory Planner
Needs:
- Stock levels
- Low stock alerts

### Regional Director
Needs:
- Sales by city
- Growth trends
