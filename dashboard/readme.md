# Dashboard Overview
-This interactive dashboard analyzes omnichannel retail sales and inventory performance across multiple cities, product categories, and sales channels. The dashboard provides business insights related to sales trends, customer purchasing behavior, inventory availability, and product performance.
-The dashboard was built using:
  -Power BI
  -SQL
  -Python (Data Cleaning & EDA)



# 💡 Final Conclusions


## This dashboard successfully provides:

-Real-time sales monitoring
-Product performance analysis
-Inventory tracking
-Regional sales comparison
-Channel-wise business insights

## The analysis helps stakeholders:

-Improve decision-making
-Optimize inventory levels
-Increase online sales performance
-Identify high-performing products and cities
-Enhance overall retail business efficiency
